# Contributing

Thanks for your interest in contributing.
I welcome every kind of contributions from bug reports to adding new code or fixing already existing code.

## Reporting issues
Before opening an issue please follow this:
 - Check if there's already existing issue
 - Include reproduction steps 
 - Include logs

## Feature requests
Please open an issue first and tell me why feature would be awesome and why exising solutions don't work.

## Commiting
Please just run `npm run lint`, then `npm run lint:fix` further run `npm run format`, and then `./test.sh` to fully test the backend and then commit piece of code.
